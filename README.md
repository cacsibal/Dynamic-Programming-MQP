# Dynamic-Programming-MQP

- A collection of dynamic programming solution for problems used in MQP
- This repository contains implementations of dynamic programming algorithms in Java

Tools:
- Java JDK 17+
- Gradle

Build: ./gradlew build

Run: ./gradlew run
